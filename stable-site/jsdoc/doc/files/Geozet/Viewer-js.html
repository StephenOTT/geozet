Geozet.Viewer

The viewer object which will be created when the DOM is ready.  All GIS components will be created in this class.

createSelectFeatureControl

createSelectFeatureControl: function()

Create the select feature control.  The select feature control is used to show popovers on mouseover and popups on click.  The events come in via the renderer.  The correct layer is dynamically attached to the SelectFeature control based on the scale of the map.
