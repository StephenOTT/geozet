Summary
Geozet.PopupA popup which will be shown on click, and a popover which will be shown on mouseover.
Properties
map{<OpenLayers.Map>} The map object to interact with.
attributes{Object} The feature attributes configuration object.
templates{Object} The templates this popup will use for presentation.
clusterZoom{Object} zoom level to zoom to if clicked on a cluster.
iconSize{Object} icon size for a set of categories.
layer{<OpenLayers.Layer.Vector>} The vector layer to interact with.
popuppedFeature{<OpenLayers.Feature.Vector>} The feature attached to this popup.
feature{<OpenLayers.Feature.Vector>} An internal reference to the feature.
popupDiv{DOMElement} The DOM element used for the popup.
popoverDiv{DOMElement} The DOM element used for the popover.
itemLength{Integer} The number of features this popup is paging.
itemCurrent{Integer} The current item which is being paged.
